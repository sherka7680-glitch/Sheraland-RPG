# 🛡️ Sheraland RPG: Character Generator 2.0

Консольное приложение на C# для генерации персонажей с системой распределения очков и проверкой логики.

### ✨ Ключевые особенности:
* **Валидация:** Используется кастомный метод для защиты от неверного ввода.
* **Логика:** Система "Алтаря" сбрасывает очки при превышении лимита.
* **Визуализация:** ASCII-графика и цветовое кодирование в консоли.
