**Sheraland RPG** — это консольная ролевая игра с открытым циклом действий, продвинутой системой генерации персонажа и элементами выживания. Весь игровой процесс завязан на управлении ресурсами: энергией, здоровьем и золотом.

---

## 🎮 Основные механики

### 🧬 Система создания героя
* **Распределение очков:** Тратьте "Магическую энергию" (статы) на Силу и Интеллект.
* **Влияние характеристик:** * `Сила` напрямую увеличивает ваше максимальное HP и вместимость инвентаря.
    * `Интеллект` увеличивает запас маны для магических классов.
* **Фактор удачи:** При создании есть 10% шанс получить статус **ЛЕГЕНДАРНЫЙ**, что дает мощный бонус к характеристикам и урону.

### ⚔️ Геймплейный цикл
В главном меню вам доступны следующие действия:
1.  **Статус:** Просмотр подробной карточки героя с визуальными HP/MP барами.
2.  **Работа:** Тратьте энергию, чтобы заработать золото.
3.  **Отдых:** Восстановление энергии за монеты.
4.  **Тренировка:** Интерактивное повышение силы через мини-игру.
5.  **Поход:** Случайные встречи — битвы с врагами (Орки, Драконы), поиск сокровищ или опасные засады.
6.  **Торговец:** Продажа найденных трофеев для пополнения кошелька.

## 🖥️ Визуальный интерфейс

Игра использует кастомную ASCII-графику для отрисовки карточки героя:

```text
┌───────────────────────────────────────┐
│ КАРТОЧКА ГЕРОЯ:                       │
├───────────────────────────────────────┤
│ Sherzod Aliev       │ Класс: Воин     │
│ Возраст: 19         │ Сложность: Hard │
├───────────────────────────────────────┤
│ HP:   150/150  ■■■■■■■■■■■■■■■····    │
│ MANA: 75/75    ■■■■■■■············    │
├───────────────────────────────────────┤
│ Сила: 12           Энергия: 8         │
│ Монеты: 45         Инвентарь: [2/11]  │
├───────────────────────────────────────┤
│ ОРУЖИЕ: ЛЕГЕНДАРНЫЙ Топор             │
└───────────────────────────────────────┘

🚀 Как запустить
Установите .NET SDK 8.0 или выше.

Создайте новый консольный проект:

Bash
dotnet new console -n SheralandRPG
Замените содержимое Program.cs кодом игры.

Запустите проект:

Bash
dotnet run
Разработчик: Sher

Проект создан в учебных целях для демонстрации работы с циклами, коллекциями и консольной графикой в C#.
